# VeroLinkr

India's escrow-secured influencer marketplace. Creators always get paid. Brands always get results.

## Tech Stack
- React 18
- Vercel (deployment)
- Supabase (database + auth) — coming in Phase 2
- Razorpay (payments) — coming in Phase 4

## Getting Started

```bash
npm install
npm start
```

## Deployment

This project is deployed automatically via Vercel.
Every push to `main` branch triggers a new deployment.
